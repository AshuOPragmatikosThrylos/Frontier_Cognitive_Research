## Abstract Ideas (bullet sketch)

- Motivation: answer-centric AI vs anomaly-centric cognition.
- Method: eighteen deterministic Python experiments, no LLM/embeddings/randomness.
- Arc: question ecology → scarcity/evolution → falsification → difference-first → memory → **competition → failed cooperation**.
- Key finding (tentative): organization and memory persist when questions are absent; **selection under budget** gates reconstruction; **simple cooperation falsified** (EXP-018).
- Limitations: behavior-diff template across categories, hand-tuned constants, no empirical validation.
- Contribution type: conceptual framework + **reproducible positive and negative results**.
