## Abstract Ideas (bullet sketch)

- Method: twenty deterministic experiments, no LLM/embeddings/randomness.
- Arc: question ecology → falsification → difference-first → memory → competition succeeds cross-domain; cooperation/merging fail.
- Key finding: **partial cross-domain generalization** — selection in 3/4 domains; CAP boundary failure.
- Layer asymmetry: questions merge (EXP-002); memory traces resist (EXP-018, EXP-019).
- Negative and boundary results as feature: program documents falsifications and limits honestly.
- Limitations: exported ranking/budget; behavior-diff template boundaries.
