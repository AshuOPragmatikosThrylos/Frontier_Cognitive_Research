# EXP-023 Results

Post-experiment summary for question removal and organization survival.

Date: 2026-06-22  
Experiment: EXP-023 Question Removal

---

## Theme

Question Removal

Test whether organizational structure — difference groups and persistent tensions — requires question objects, question IDs, question states, memory, or fresh emergence; or survives with differences and tensions alone.

---

## Experiment Summary

Started from:

```
observations → differences → tensions
```

Two worlds received **identical observations** through difference formation and tension detection. Worlds diverged at the question layer.

### World A — Questions Enabled

| Property | Behavior |
|----------|----------|
| Question lifecycle | Emerge → resolve → extinct → delete (no memory traces) |
| Reintroduction | Fresh emergence from reopened tensions |
| Memory / reconstruction | **None** |

**Outcome:** Three questions cycled through lifecycle; three fresh emergences on reintroduction; organization intact.

### World B — No Question Layer

| Removed | Detail |
|---------|--------|
| Question objects | No `LifecycleQuestion` instances created |
| Question IDs | None |
| Question states | None |
| Memory | None |
| Reconstruction | None |
| Fresh emergence | None |

**Outcome:** Zero questions; six difference groups; three persistent tensions; organization intact.

### Shared outcome

**Organization survived in both worlds.**

World B maintained **difference groups** and **persistent tensions** without any question layer:

| Layer | World A | World B |
|-------|---------|---------|
| Observations | 28 | 28 |
| Difference groups | 6 | 6 |
| Persistent tensions | 3 | 3 |
| Live questions | 3 | 0 |
| Organization survives | Yes | **Yes** |

### Conclusion

The experiment demonstrated that **questions are optional and are not required for organization**. Tensions alone maintain structure when the question compression layer is removed entirely.

---

## Supported Hypotheses

| Hypothesis | Statement | Evidence |
|------------|-----------|----------|
| **H187** | Questions are not fundamental | World B fully organized with zero question objects |
| **H188** | Tensions alone maintain organization | World B: 6 groups, 3 persistent tensions, no questions |
| **H189** | Questions are expressions of tensions | World A questions map 1:1 to tensions; World B needs no separate expression layer for structure |
| **H190** | Questions are epiphenomena | Organization invariant with or without question layer |
| **H191** | Questions are compressed representations of tensions | Same tension ids drive question text in World A; structure exists without compression in World B |
| **H192** | Objects are conveniences rather than necessities | Question objects optional; difference+tension sufficient |
| **H193** | Organization precedes representation | Differences and tensions precede and survive without question representation |

---

## Unexpected Observations

1. **Organization survived after removing questions.** No collapse at difference or tension layers.

2. **Persistent tensions remained.** All three tensions reopened with identical strength profiles in both worlds.

3. **Question IDs and states proved unnecessary.** World B operated with no question identifiers or lifecycle states.

---

## Surprising Observations

1. **The project named Question-Centric Intelligence successfully removed questions.** The program's namesake primitive was falsified as structural requirement while the program remained coherent.

2. **The ecosystem retained structure without any question layer.** Six difference groups and three tensions — identical counts in World A and World B after reintroduction.

3. **Questions appeared to be derived representations rather than fundamental entities.** World A compresses tensions into live questions; World B shows tensions alone suffice for organization.

## Conclusion

EXP-023 compared a question-enabled ecosystem with a tension-only world. Organization survived in both. World B maintained difference groups and persistent tensions without questions, IDs, states, memory, or emergence.

Questions are not fundamental (H187). Tensions alone maintain organization (H188). Questions express, compress, or epiphenomenally accompany tensions (H189–H191). Objects are conveniences (H192); organization precedes representation (H193).

The Question-Centric Intelligence program successfully removed its namesake layer while preserving structure. See `exp023_failures.md` for remaining dependencies on tensions and differences.
