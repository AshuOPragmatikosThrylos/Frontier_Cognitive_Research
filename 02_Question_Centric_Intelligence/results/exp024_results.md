# EXP-024 Results

Post-experiment summary for tension removal and organization survival.

Date: 2026-06-22  
Experiment: EXP-024 Tension Removal

---

## Theme

Tension Removal

Test whether organizational structure — difference groups — requires tension objects, tension IDs, strength tracking, or persistent tensions; or survives with differences alone.

---

## Experiment Summary

Started from:

```
observations → differences
```

Two worlds received **identical observations** through difference formation. Worlds diverged at the tension layer.

### World A — Tensions Enabled

| Property | Behavior |
|----------|----------|
| Difference groups | Formed per category when ≥2 behaviors with ≥2 members each |
| Persistent tensions | Detected between paired groups; strength = min member counts |
| Questions | **None** |
| Memory | **None** |
| Persistence / reconstruction | **None** |

**Outcome:** Six difference groups; three persistent tensions after reintroduction; organization intact.

### World B — No Tension Layer

| Removed | Detail |
|---------|--------|
| Tension objects | No `PersistentTension` instances created |
| Tension IDs | None |
| Tension strength | None |
| Persistent tensions | None |
| Questions | None |
| Memory | None |
| Persistence / reconstruction | None |

**Outcome:** Zero tensions; six difference groups; organization intact.

### Shared outcome

**Organization survived in both worlds.**

World B maintained **difference groups** without any tension layer:

| Layer | World A | World B |
|-------|---------|---------|
| Observations | 28 | 28 |
| Difference groups | 6 | 6 |
| Persistent tensions | 3 | 0 |
| Questions | 0 | 0 |
| Organization survives | Yes | **Yes** |

### Conclusion

The experiment demonstrated that **tensions are optional and are not required for organization**. Difference groups alone maintain structure when the tension layer is removed entirely.

| **H200** | Organization precedes conflict | Difference groups precede and survive without tension representation |
| **H201** | Difference groups are sufficient | World B: 6 groups, 0 tensions, organization survives |

## Hypotheses Not Supported

| Hypothesis | Statement | Evidence |
|------------|-----------|----------|
| **H194** | Tension is fundamental | World B fully organized without any tension layer |
| **H195** | Differences alone cannot organize | World B: six active difference groups, organization survives |
| **H196** | Tension is the first irreducible structure | Tension layer removed; structure intact at difference floor |
| **H197** | Removing tension destroys organization | No collapse in World B |

---

## Unexpected Observations

1. **Organization survived after removing tensions.** No collapse at the difference layer.

2. **Persistent tensions proved unnecessary.** World B operated with no tension identifiers, strength tracking, or persistent conflict objects.

3. **No higher layers were required.** Neither questions, memory, persistence, nor reconstruction needed in either world.

---

## Surprising Observations

1. **Differences alone maintained structure.** Six groups in both worlds with identical member counts after reintroduction.

2. **Tensions appeared to be optional summaries rather than fundamental entities.** World A pairs and ranks group conflicts; World B shows difference groups alone suffice for organization.

3. **The ecosystem retained organization without conflict representation.** World B logs active difference groups; no tension objects at any stage.

## Conclusion

EXP-024 compared a tension-enabled ecosystem with a difference-only world. Organization survived in both. World B maintained difference groups without tensions, tension IDs, strength, questions, memory, or reconstruction.

Tensions are not fundamental (H194 falsified). Differences alone maintain organization (H198). Tensions are pairwise summaries of differences (H199). Organization precedes conflict representation (H200). Difference groups are sufficient (H201).

The destructive arc has now removed four layers — selection, persistence, questions, and tensions — while structure persists at the difference floor. See `exp024_failures.md` for remaining dependency on differences.
