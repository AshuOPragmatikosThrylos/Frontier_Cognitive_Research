# EXP-025 Results

Post-experiment summary for difference removal and organizational collapse.

Date: 2026-06-22  
Experiment: EXP-025 Difference Removal

---

## Theme

Difference Removal

Test whether organizational structure requires difference groups, labels, categories, partitions, and distinction objects; or survives with raw observations alone.

---

## Experiment Summary

Started from:

```
observations
```

Two worlds received **identical entity streams**. Worlds diverged at the difference layer.

### World A — Difference Groups Enabled

| Property | Behavior |
|----------|----------|
| Difference groups | Formed per category when ≥2 behaviors with ≥2 members each |
| Tensions | **None** |
| Questions | **None** |
| Memory | **None** |
| Persistence / reconstruction | **None** |

**Outcome:** Six difference groups after reintroduction; organization intact.

### World B — Raw Observations Only

| Removed | Detail |
|---------|--------|
| Difference groups | None formed |
| Labels | None |
| Categories | None indexed |
| Partitions | None |
| Distinction objects | None |
| Tensions | None |
| Questions | None |
| Memory | None |
| Persistence / reconstruction | None |
| Competition / selection | None |

**Outcome:** Twenty-eight flat entity entries; zero groups; organization collapsed.

### Contrasting outcome

**Organization survived in World A. Organization collapsed in World B.**

| Layer | World A | World B |
|-------|---------|---------|
| Observations | 28 | 28 |
| Difference groups | 6 | **0** |
| Categories | 3 | **0** |
| Partitions | 6 | **0** |
| Organization survives | **Yes** | **No** |

### Conclusion

The experiment demonstrated that **raw observations alone are insufficient**. **Difference groups were necessary for organization.**

## Unexpected Observations

1. **The reduction program reached collapse for the first time.** EXP-021–024 removed layers while organization persisted; EXP-025 produced the first clean failure.

2. **Raw observations formed no structure.** Identical entity count in both worlds; only World A produced partitions.

3. **Difference groups maintained organization without higher layers.** Six groups sufficient with no tensions, questions, memory, or persistence.

---

## Surprising Observations

1. **Everything above differences proved optional.** Selection, persistence, questions, and tensions all removed in prior experiments without collapse.

2. **Difference survived repeated attacks.** EXP-024 confirmed difference-only organization; EXP-025 confirmed difference necessity.

3. **Organization failed only when differences disappeared.** First asymmetric outcome in the destructive arc — survival until bedrock, then collapse.
