# Compression Failure

## Definition

A mismatch between the world model prediction and observed reality.

The world model expects certain outcomes. When an observation does not match that expectation, compression fails — the observation cannot be cleanly absorbed into the current model.

Compression failures are hypothesized to be the source of curiosity.

## Example: Bird → Fly

**World model expectation:** Birds fly.

**Observation:** Penguin does not fly.

**Compression failure:** The observation violates the model rule that birds fly.

**Question:** Why don't penguins fly?

A new question is born from the failure to compress the observation into the existing world model.
